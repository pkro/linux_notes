#### Time commands

- `time [command]` measures time consumed by a process (bash built-in), e.g. `time find / -name core`

:~/$ time find / -name core
[...command output]
real    0m23,043s # real time (duh)
user    0m1,599s # cpu time used
sys     0m5,852s # time from kernel supporting the process
